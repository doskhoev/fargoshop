# FargoShop - Интернет-магазин продуктов питания

Современный интернет-магазин продуктов питания, построенный на Next.js, TypeScript и Tailwind CSS.

## Функциональность

- 🛒 Каталог товаров с фильтрацией
- 🛍️ Корзина покупок с управлением количеством
- 📱 Адаптивный дизайн
- 💾 Сохранение корзины в localStorage
- 🎨 Современный UI с Tailwind CSS

## Технологии

- **Frontend:** Next.js 14, React 18, TypeScript
- **Стилизация:** Tailwind CSS
- **Управление состоянием:** Zustand
- **Иконки:** Heroicons
- **Пакетный менеджер:** Yarn

## Установка и запуск

1. Установите зависимости:
```bash
yarn install
```

2. Запустите проект в режиме разработки:
```bash
yarn dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Доступные команды

- `yarn dev` - запуск в режиме разработки
- `yarn build` - сборка для продакшена
- `yarn start` - запуск продакшен сборки
- `yarn lint` - проверка кода линтером

## Структура проекта

```
src/
├── app/                 # App Router (Next.js 13+)
├── components/          # React компоненты
├── store/              # Zustand store
├── types/              # TypeScript типы
├── data/               # Мокапы данных
└── styles/             # Глобальные стили
```
